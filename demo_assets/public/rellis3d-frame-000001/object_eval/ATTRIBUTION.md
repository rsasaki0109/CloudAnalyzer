This directory contains JSON examples derived from the public RELLIS-3D seed
frame stored one level up in `demo_assets/public/rellis3d-frame-000001`.

- The detection JSONs are derived from semantic label clusters on that public
  frame.
- The tracking JSONs are deterministic synthetic 3-frame sequences seeded from
  those derived boxes.

These files and any derivative examples continue to follow the upstream dataset
terms.
